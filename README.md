# MicroSkills Landing

Landingpage für MicroSkills — SKL-Prüfungsvorbereitung für Physiotherapie im DACH-Raum.

## Deployment

Statische HTML-Seiten, optimiert für Coolify-Deployment.

## Struktur

- `index.html` — Hauptseite
- `app.html` — App / Trial-Registrierung
- `paywall.html` — Paywall / Stripe-Checkout
- `success.html` — Stripe Erfolgsseite
- `agb.html`, `datenschutz.html`, `impressum.html` — Rechtliches
- `robots.txt`, `sitemap.xml` — SEO
- `*-staatsexamen.html` — Subject-Landingpages mit Email-Modal
